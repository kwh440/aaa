# 🧮 Assignment: Student Score Analyzer

## 💻 Environment: Code::Blocks (C language)  
## 🔗 Submission: Push to your GitHub Classroom repository by 04/08/2025 4,00 pm

## 🎯 Objective
Build a **menu-driven program** in C to analyze student scores. Use **arrays**, **functions**, and **basic statistics** to:
- Input N student marks (1–100)
- Find **total**, **average**, **highest**, and **lowest** marks
- Calculate **median** and **standard deviation**
- Assign **grades** based on score boundaries
- Draw an ASCII bar graph of grades.
- Predict next score using a moving average.

## 📌 Menu Structure

1. Input scores  
2. Display total and average  
3. Display highest and lowest scores  
4. Display median  
5. Display standard deviation  
6. Show grades  
7. Exit
